# Daten-Verzeichnis

Dieses Verzeichnis enthält:
- `uploads/` - Hochgeladene Excel- und PDF-Dateien
- `exports/` - Generierte Rechnungen und Reports
- `sample/` - Beispieldateien für Tests

## Verzeichnisstruktur
```
data/
├── uploads/
│   ├── excel/
│   └── pdf/
├── exports/
│   ├── invoices/
│   └── reports/
└── sample/
    └── test_data.xlsx
```
